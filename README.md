# WorldEdit-SE
WorldEdit SE(Special Edition) 2020はpmmpのための最適化されたWorldEditプラグインです。

## 機能

以下の機能があります or 実装予定です。

| 機能名                         | 対応          |
| -------------------------------------------| ------------------------|
| 非同期処理                | 予定   |
| ユーザー別タスク管理                | 予定   |
| 直方体型敷き詰め               | 予定    |
| 球型敷き詰め               | 予定    |
| 円柱型敷き詰め                   | 予定     |
| 置き換え(replace)                  | 予定    |
| 取り消し(undo)                   | 予定      |
| 取り消しを取り消し??(redo)                   | 予定     |
| 切り取り(cut) | 予定      |
| 貼り付け(paste) | 予定      |
| 入力(Import from JSON)                   | 予定     |
| 出力(Export as JSON) | 予定      |
| パーティクルプレビュー表示              | 予定      |
| ベンチマーキング                  | 予定      |
| 推測完了時間(ETA)の表示               | 予定       |
| ユーザーが埋まらないように                                     |  予定       |
